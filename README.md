## Estrutura

```
portfolio/
└── index.html        # App completo — HTML + CSS + JS inline
    ├── Navbar
    ├── Hero
    ├── Projetos (grid de cards)
    ├── Sobre mim
    └── Contato
```

Arquivo único, zero dependências de build. Basta abrir no browser ou fazer deploy de qualquer plataforma de hosting estático.

---

## Como rodar localmente

```bash
# Clonar
git clone https://github.com/LuisGuilhermeRampaso/portfolio.git
cd portfolio

# Abrir direto no browser
start index.html   # Windows
open index.html    # macOS
```

Ou use qualquer servidor estático:

```bash
npx serve .
# → http://localhost:3000
```
